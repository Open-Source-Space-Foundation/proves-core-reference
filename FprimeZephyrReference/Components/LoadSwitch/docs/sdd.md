## Port Descriptions

|---|---|
| Status | Boolean value displaying whether the load switch is on or not. |

## Component States
| Name  | Description                          |
|-------|--------------------------------------|
| Off   | No power to the component            |
| On    | Power supplied to the component      |
| Error | An error occurred in the load switch |
